# ElsaTech — Static site for GitHub Pages

This repository contains a minimal, modern static website for ElsaTech.

## Files
- `index.html` — main site
- `css/style.css` — styles
- `js/main.js` — tiny interactions
- `assets/logo.png` — replace this with your logo file (you uploaded one in the chat; add same filename here)

## Quick setup for GitHub Pages
1. Create a new repository on GitHub (e.g. `elsatech` or `elsatech-website`).
2. Clone locally and copy files (or upload in GitHub web UI):
   - index.html
   - css/style.css
   - js/main.js
   - assets/logo.png
3. Commit and push.
4. In the repo settings -> Pages, select `main` branch and `/ (root)` folder. Save.
5. After a minute your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Notes
- If your logo filename or path differs, update `src="assets/logo.png"` in `index.html`.
- The contact form uses a `mailto:` action — for a production site wire up a backend or a form provider.
